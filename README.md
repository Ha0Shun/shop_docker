# Django Shopping Website

這是基於 Django 架構的購物網站，利用 Docker 和 Nginx 進行部署。此專案旨在展示如何建立一個簡單的購物網站，並使用容器化技術將其部署到生產環境。**本網站僅供測試使用，無任何營利行為。**

## 功能

此購物網站提供基本的購物功能，使用戶能夠進行商品瀏覽、加入購物車、結帳等操作。主要功能包括：

- **商品瀏覽**：用戶可以查看網站上的所有商品，並能夠查看商品詳情。
- **購物車功能**：用戶可以將商品添加到購物車、修改商品數量或刪除商品。
- **結帳流程**：用戶可以利用綠界金流（ECPay）測試版完成訂單的支付。
- **訂單查詢**：用戶可以查看過去的訂單，並追蹤訂單狀態。
- **後台管理系統**：使用 Django 內建的管理後台管理商品、訂單、用戶等資料。

## 使用的技術

- **Django** - 用於網站的後端開發
- **Nginx** - 作為反向代理伺服器和靜態文件伺服器
- **Docker** - 用於容器化部署
- **Linux** - 運行伺服器的作業系統
- **MySQL** - 作為網站的資料庫，負責存儲用戶、商品、訂單等資料
- **綠界金流（ECPay）測試版** - 用於處理線上支付
